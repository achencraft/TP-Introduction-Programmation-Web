Les balises d'en-tête

  1. Les balises <meta>

    • Description : Cette balise permet aux moteurs de recherche de connaitre puis d'afficher dans les résultats la description du site, redigée par nos soins.
      Elle s'utilise de cette manière : <meta name="description" content="la desc. du site">
    • Keywords : Cette balise permet de fournir aux moteurs de recherche des mots clés concernants le site.
      Cela permet de faciliter la recherche du site sur Internet
      Elle s'utilise de cette manière : <meta name="keywords" content="tramway, strasbourg, cts">
    • Author : Cette balise permet de fournir aux moteurs de recherche l'auteur d'une ou des pages du site
      Elle s'utilise de cette manière : <meta name="author" content="Lucas LETT">
  2. Pas de question ici

  3. Les feuilles de style

    La balise <link> permet d'integrer une feuille de style externe à la page html.
    Elle s'integre comme ceci : <link rel="stylesheet" href="../../style/style.css" />
    Il est interessant d'utiliser des balises locales <style> lorsque la modification de style est isolée, qu'elle ne touche qu'un seul élement.

  4. La balise meta Viewport

    Cette balise permet de creer des sites responsive.
    Cela signifie que les appareils mobiles comme les smartphones ou les tablettes sauront comment zoomer afin d'afficher la page html de manière lisible.
    Elle est utilisée de cette manière : <meta name="viewport" content="width=device-width">

  5. Le type d'encodage

    L'encodage utf-8, appelé Unicode, est un encodage regroupant la totalité des caractères de toutes les langues écrites.
    Il permet donc de proposer des textes pour toutes les langues, sans avoir à changer d'encodage.
    Cette page est écrite en Utf-8 et gère donc les accents, contrairement à l'encodage standard américain ASCII par exemple.

Mise en forme de texte avec CSS

  1. mise en page des balises h2 de classe avecserif

    h2.avecserif{
    font-family:Times New Roman;
    font-size:120%;
    }
  2. J'utilise des taille de police relatives, plutôt d'absolue, pour que le texte reste lisible si on change le zoom du navigateur, par exemple.

  3. mise en page des balises p

    p{
    color:#404040;
    font-family:sans-serif;
    }

Les formulaires

  1. la pseudo-classe :hover

    Cette pseudo-classe précise comment doit s'afficher un objet lorsque le curseur de souris le survole !
    Il existe d'autres pseudo-classes comme :link :visited :active ...

  2. la propriété box-shadow

    Cette propriété affiche l'ombre d'un objet.
    Elle s'utilise ainsi :
    /* décalage-x | décalage-y | rayon_de_flou | étalement | couleur */
    box-shadow: 2px 2px 2px 1px rgba(0, 0, 0, 0.2);

La propriété float du CSS

  1. La propriété float va faire sortir l'élément équipé du "flux" standard. Il va donc y avoir des superpositions.
  2. Pour eviter ces superpositions, je vais mettre les éléments voisins en float aussi.
    Je vais même utiliser la propriété clear !
  3. Je vais utiliser "clear:both;" pour le footer, afin qu'il se place SOUS tous les éléments flottant.
    Il existe aussi "clear:left;" et "clear:right;".

Premiers pas en Javascript

  1. Accès à un élement

    J'utilise la ligne : document.getElementById('montexte')
    on peut aussi utiliser getElementByTagName() ou encore getElementByName().

  2. Editer le contenu d'un élement

    J'utilise la ligne : document.getElementById('montexte').innerHTML = 'nouveau texte';

Les objets en Javascript

  1. Les attributs et les méthodes

    Les attributs sont les caracteristiques d'un objet, comme la massa ou la couleur puevent l'être pour une voiture.
    Les méthodes sont les fonctions pas lesquelles on va pouvoir acceder aux valeurs des attributs et les utiliser.

  2. Creer un objet

    Il faut utiliser un constructeur, avec la cmd new.

  3. Accès aux données

    via objet_name.attribut
    ou objet_name['attribut']

Les écouteurs d'événements

  1. x.addEventListener(“mouseover”, fonction1) va effectuer la fonction1 si la souris survole l'élément x.

  2. mouseover et mouseout détectent l'entrée et la sortie de la souris sur un élément.

  3. document.getElementsByClassName(name) va récuperer les éléments du DOM appartenant à une classe donnée.

  4. x.style.backgroundColor ="#ff0" va changer la couleur de fond de l'élément x.

  5. position va sortir la description du flux pour pouvoir la placer sur l'image.
    display va par defaut cacher la description
    bottom va la placer à une distance du bas de l'image correpondant à 10% de sa propre taille
    left va coller la description à gauche
